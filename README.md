# 🏫 우리반 프리미엄 학급홈페이지 (Next.js)

우리의 꿈과 추억이 자라나는 공간, 넥스트 제이에스(Next.js) 기반의 고성능 학급 홈페이지입니다.

## ✨ 주요 기능
- **프리미엄 디자인:** 글래스모피즘(Glassmorphism)과 다이내믹 그라데이션이 적용된 세련된 UI
- **공지사항:** 학급의 중요한 소식을 한눈에 확인
- **갤러리:** 학급 활동 사진을 볼 수 있는 반응형 그리드
- **시간표:** 오늘의 일정을 시각적으로 표현한 타임라인

## 🚀 기술 스택
- **Framework:** Next.js 15+ (App Router)
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Styling:** CSS Modules
- **Deployment:** Vercel (Ready)

## 🛠 실행 방법

```bash
# 의존성 설치
npm install

# 로컬 개발 서버 실행
npm run dev
```
